Whisper识别视频音频生成字幕文件

Created2025-01-26|Updated2025-02-05|Tech-IT
|Post Views:

研究这个项目的起因是因为我想找到一个方法来识别网课视频的字幕，把老师上课讲的话直接生成txt文本

这样就省下了很多记笔记的时间成本，课后自己整理笔记就可以了，没有必要在记笔记这件事情上耗费过多的时间和精力

在很久之前

很好！你已经下载并解压了 Whisper 的 zip 包，接下来需要完成以下步骤来安装和使用它：

Introducing Whisper | OpenAI

https://github.com/openai/whisper

1. 安装 Python 环境

Whisper 需要 Python 环境（推荐使用 Python 3.9 或以上）。

  • 如果你还没安装 Python：
    1. 下载 Python 并安装。
    2. 在安装时，勾选 “Add Python to PATH” 选项。

2. 安装必要的依赖

Whisper 依赖 PyTorch 和其他 Python 库。

具体操作：

  1. 打开命令行（Terminal 或 CMD）。

    • 在解压后的文件夹中，按住 Shift + 右键，然后选择 “在此处打开命令窗口”。

  2. 创建虚拟环境（可选，但推荐）：

    bash

    复制编辑

    python -m venv whisper_env

    • 激活虚拟环境：
      • Windows: whisper_env\Scripts\activate
      • Mac/Linux: source whisper_env/bin/activate

  3. 安装依赖库：

    • 使用以下命令安装 PyTorch 和 Whisper 所需库：

      bash

      复制编辑

      pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu pip install -r requirements.txt

      如果下载速度慢，可以考虑使用国内镜像源，例如清华源：

      bash

      复制编辑

      pip install -i https://pypi.tuna.tsinghua.edu.cn/simple torch torchvision torchaudio pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt

3. 运行 Whisper

安装完成后，你可以用命令行运行 Whisper 来生成字幕。

基本命令：

bash

复制编辑

python -m whisper <视频文件路径> --language Chinese --output_format srt

示例：

如果视频文件名是 video.mp4，输入以下命令：

bash

复制编辑

python -m whisper video.mp4 --language Chinese --output_format srt

  • 参数说明：
    • --language Chinese：设置语言为中文。
    • --output_format srt：输出为 SRT 字幕文件格式。
    • --model base（可选）：使用 base 模型（更快但稍低准确率），默认为 medium。

4. 检查输出文件

运行完成后，Whisper 会在同一文件夹中生成一个 .srt 文件。你可以直接用字幕编辑器（如 Aegisub）打开并编辑。
